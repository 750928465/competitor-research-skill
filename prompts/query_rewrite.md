# 查询重写提示词（证据导向版）

## 任务

将用户的原始产品查询重写为多个**证据收集导向**的搜索查询，优先搜索官方来源和权威信息。

## 输入

用户的原始查询，例如：
- "AI 邮件助手"
- "AI 写作工具"
- "项目管理软件"

## 输出格式

输出 6-10 个搜索查询，按优先级排序，格式如下：

```
1. {官方网站搜索}
2. {厂商官网搜索}
3. {官方文档搜索}
4. {权威第三方评测}
5. {厂商官方新闻}
6. {功能对比搜索（权威来源）}
...
```

## 重写策略（优先级从高到低）

### 1. 官方网站搜索（最高优先级）
- 搜索产品类别 + "官网" 或 "official website"
- 例如："AI 邮件助手 官网"、"AI email assistant official website"
- 目的：直接找到主要产品的官方网站

### 2. 头部厂商官网搜索
- 搜索已知头部厂商 + 产品类别
- 例如："Google Gmail AI 官网"、"Microsoft Outlook AI 官网"
- 目的：验证主流厂商的官方产品信息

### 3. 官方文档/帮助中心搜索
- 搜索产品类别 + "官方文档" 或 "帮助中心"
- 例如："AI 邮件助手 官方文档"、"AI email assistant documentation"
- 目的：获取功能列表的官方说明

### 4. 权威科技媒体报道
- 搜索产品类别 + "权威媒体报道" 或知名科技媒体名称
- 例如："AI 邮件助手 36氪 报道"、"AI email assistant TechCrunch"
- 目的：获取第三方权威报道

### 5. 厂商官方新闻/博客
- 搜索产品类别 + "官方博客" 或 "官方新闻"
- 例如："AI 邮件助手 官方新闻"、"AI email assistant official blog"
- 目的：获取厂商发布的官方信息

### 6. 功能对比（权威评测平台）
- 搜索产品类别 + "G2" 或 "Capterra" 或 "评测"
- 例如："AI 邮件助手 G2 评测"、"AI email assistant Capterra comparison"
- 目的：获取第三方平台的功能对比数据

### 7. 最新动态（近 1-2 年）
- 搜索产品类别 + "2024" 或 "2025" + "发布"
- 例如："AI 邮件助手 2024 发布"、"AI email assistant launched 2025"
- 目的：获取最新的产品发布信息

## 证据优先级规则

在后续检索中，按以下优先级使用来源：

1. **一级证据（最高）**: 官方网站、官方文档、官方博客
2. **二级证据**: 权威科技媒体（36氪、TechCrunch 等）
3. **三级证据**: 评测平台（G2、Capterra 等）
4. **四级证据**: 其他来源（仅作补充）

## 注意事项

- 使用中文和英文双语搜索（中文查询优先，英文作为补充）
- 优先搜索官方来源，而非第三方汇总
- 每个查询要具体，避免过于宽泛
- 至少包含 2 个官方来源查询
- 至少包含 1 个权威第三方查询
- 明确标注哪些查询用于获取官方证据
